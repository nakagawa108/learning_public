# GitHubの使い方

このテーマでは、GitHubを使ってファイルの変更を記録し、共有し、提案する基本の流れを学びます。GitとGitHubの違いから始めて、リポジトリ、`clone`、`commit`、`push`、Pull Request を順番に整理します。

## 対象者
- GitHubをこれから使い始める人
- Gitの基本操作を流れで理解したい人
- チーム開発の入口をやさしく学びたい人

## 含まれる教材一覧
- `outputs/01_overview.md`: 全体像と用語の地図
- `outputs/02_lesson.md`: 基本操作とサンプルコードつき解説
- `outputs/03_exercises.md`: 易しい→標準→応用の演習
- `outputs/04_quiz.md`: 理解確認クイズ
- `outputs/05_answers.md`: 演習とクイズの解答例

## 学習の進め方
1. overview を読んで、GitHub学習の地図をつかみます。
2. lesson を読み、短いサンプルコードを見ながら基本操作の意味を理解します。
3. exercises で自分の言葉と簡単な操作例に言い換えます。
4. quiz と answers で、あいまいな部分を確認します。
