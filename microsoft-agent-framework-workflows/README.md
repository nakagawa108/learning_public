# Microsoft Agent Framework Workflows

このテーマでは、Microsoft Agent Framework の `Workflow` 機能の使い方を重点的に学びます。  
単発の Agent 実行ではなく、「複数ステップを順番に流す」「条件で分岐する」「人の確認を途中に入れる」といった設計を理解するための教材です。

## 対象者
- Microsoft Agent Framework を学び始めた人
- Agent はわかるが Workflow の必要性がまだピンと来ない人
- Python のコード例を読みながら段階的に理解したい人

## 含まれる教材一覧
- `outputs/01_overview.md`: 全体像と学習順
- `outputs/02_lesson.md`: Workflow の詳しい解説
- `outputs/03_exercises.md`: 段階別の演習
- `outputs/04_quiz.md`: 理解確認クイズ
- `outputs/05_answers.md`: 演習とクイズの解答
- `source/notes.md`: 参照元メモ

## 学習の進め方
1. まず `01_overview.md` で Workflow の位置づけをつかむ
2. 次に `02_lesson.md` を上から順に読み、最小例から分岐・並列・HITL へ進む
3. `03_exercises.md` で手を動かす
4. `04_quiz.md` と `05_answers.md` で理解の穴を埋める
