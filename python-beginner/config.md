# Python 初級 教材設計

## テーマ名
Python 初級

## 想定読者
- プログラミングをこれから学び始める初学者
- Python を使って最初のコードを書いてみたい人

## 学習ゴール
- Python で値を表示し、変数にデータを入れられる
- 条件分岐と繰り返しの基本を使って簡単な処理を書ける
- 関数を使って処理を分けられる
- リストと辞書の基本を理解し、簡単なデータ操作ができる
- エラーを怖がりすぎず、少しずつ読み解く姿勢を持てる

## 文体
- 完全な初学者にも読めるように、短い文でやさしく説明する
- 専門用語は最初に短く定義する
- 例を先に示し、そのあとに意味を説明する
- 1 セクションで 1 つの概念に絞る

## 禁止事項
- 中級以上の話題へ大きく広げすぎない
- オブジェクト指向、デコレータ、ジェネレータなどへ寄り道しない
- 抽象的な説明だけで終わらせない
- 初学者に必要ない開発環境や内部仕様の話を中心にしない
