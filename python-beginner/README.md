# Python 初級

このテーマでは、Python を初めて学ぶ人向けに、基本の書き方を順番に学びます。変数、条件分岐、繰り返し、関数、リスト、辞書などを通して、簡単なプログラムを自分で読んで書ける状態を目指します。

## 対象者
- プログラミングがほぼ初めての人
- Python で最初の一歩を踏み出したい人

## 含まれる教材一覧
- `outputs/01_overview.md`: 学習全体の見取り図
- `outputs/02_lesson.md`: 基本概念とコード例つきの本編
- `outputs/03_exercises.md`: 易しい問題から始める演習
- `outputs/04_quiz.md`: 理解確認クイズ
- `outputs/05_answers.md`: 演習とクイズの解答

## 学習の進め方
1. まず overview を読んで、何を学ぶのか全体像をつかみます。
2. 次に lesson を読み、短いコードをまねしながら意味を確認します。
3. exercises で手を動かし、quiz で理解を確認します。
4. 最後に answers を見て、自分の答えと比べながら復習します。
