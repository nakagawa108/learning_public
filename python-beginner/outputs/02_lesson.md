# Python 初級 レッスン

## 1. `print()` で表示する
最初に覚えるのは、画面に文字や数字を表示する方法です。

```python
print("Hello, Python")
print(3)
```

`print()` は、中に入れた値を画面に表示します。

文字は `" "` で囲みます。数字はそのまま書けます。

### 要点
- `print()` は表示の基本です。
- 文字列は `" "` で囲みます。

## 2. 変数とは何か
変数とは、値に名前をつけて覚えておくためのものです。

```python
name = "Aki"
age = 20

print(name)
print(age)
```

このコードでは、`name` と `age` という名前で値を保存しています。

変数があると、同じ値を何度も使いやすくなります。

### 要点
- 変数は、値を覚えておくための名前つきの箱です。
- `=` は「右の値を左の名前に入れる」と考えるとわかりやすいです。

## 3. 計算してみる
Python では数字の計算もできます。

```python
price = 100
count = 3
total = price * count

print(total)
```

このコードでは、`100 * 3` の結果を `total` に入れて表示しています。

### 要点
- 計算結果も変数に入れられます。
- `+`、`-`、`*`、`/` などの演算子が使えます。

## 4. `if` で場合分けする
条件によって動きを変えたいときは `if` を使います。

```python
score = 80

if score >= 60:
    print("pass")
```

このコードでは、`score` が `60` 以上のときだけ `pass` を表示します。

場合分けを増やしたいときは `else` も使えます。

```python
score = 40

if score >= 60:
    print("pass")
else:
    print("fail")
```

インデントされた部分が、その条件で実行される処理です。

### 要点
- `if` は条件が合うときだけ処理を行います。
- `else` は、それ以外の場合の処理です。

## 5. `for` で繰り返す
同じような処理を何度も行いたいときは `for` を使います。

```python
for i in range(3):
    print(i)
```

このコードは、`0`、`1`、`2` を順番に表示します。

`range(3)` は、3 回分の数字を順番に作るものと考えるとわかりやすいです。

リストの中身を順番に見ることもできます。

```python
fruits = ["apple", "banana", "orange"]

for fruit in fruits:
    print(fruit)
```

### 要点
- `for` は繰り返し処理の基本です。
- `range()` やリストと一緒によく使います。

## 6. 関数で処理をまとめる
関数とは、処理に名前をつけてまとめる仕組みです。

```python
def greet():
    print("Hello")

greet()
```

`def` で関数を作り、あとで関数名を書いて呼び出します。

値を受け取ることもできます。

```python
def greet(name):
    print(f"Hello, {name}")

greet("Aki")
greet("Mio")
```

同じ形の処理をくり返し使いたいときに便利です。

### 要点
- 関数は処理をまとめて再利用する仕組みです。
- 引数を使うと、入れる値だけ変えて使えます。

## 7. リストを使う
リストとは、複数の値を順番にまとめるデータです。

```python
scores = [80, 90, 70]

print(scores)
print(scores[0])
```

`scores[0]` は最初の値を取り出しています。

要素を追加することもできます。

```python
scores = [80, 90]
scores.append(70)

print(scores)
```

### 要点
- リストは複数の値をまとめるときに使います。
- `[0]` は最初の要素を取り出します。

## 8. 辞書を使う
辞書とは、名前と値を組み合わせて保存するデータです。

```python
user = {"name": "Aki", "age": 20}

print(user["name"])
print(user["age"])
```

このコードでは、`name` や `age` を手がかりに値を取り出しています。

### 要点
- 辞書は「項目名: 値」の形でデータを持ちます。
- 名前を使って値を取り出せます。

## 9. 入力を受け取る
ユーザーから文字を受け取りたいときは `input()` を使います。

```python
name = input("Your name: ")
print(f"Hello, {name}")
```

`input()` の結果は文字列として入ります。

数字として使いたいときは変換が必要です。

```python
age = int(input("Your age: "))
print(age + 1)
```

### 要点
- `input()` は入力を受け取ります。
- 数字として使うときは `int()` などで変換します。

## 10. よくあるミス
初学者がしやすいミスをいくつか見ておきます。

### ミス 1. 文字列を `" "` で囲まない

```python
print(Hello)
```

これは `Hello` を変数名として読もうとするので、エラーになります。

正しくは次のように書きます。

```python
print("Hello")
```

### ミス 2. インデントがずれる

```python
if 3 > 1:
print("ok")
```

`if` の中の処理は、字下げが必要です。

```python
if 3 > 1:
    print("ok")
```

### ミス 3. 変数名をまちがえる

```python
name = "Aki"
print(nmae)
```

変数名のつづりが違うとエラーになります。

### 要点
- 初学者のミスは珍しくありません。
- エラーメッセージを見ながら 1 つずつ直せば大丈夫です。

## まとめ
- `print()` で表示できます。
- 変数で値を覚えられます。
- `if` で場合分けし、`for` で繰り返せます。
- 関数で処理をまとめられます。
- リストと辞書で複数のデータを扱えます。
