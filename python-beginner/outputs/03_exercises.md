# Python 初級 演習

## 演習 1. 易しい
次のコードは何を表示しますか。

```python

print(name)
```

## 演習 2. 易しい
次のコードで、`total` には何が入りますか。

```python
price = 200
count = 4
total = price * count
```

## 演習 3. 易しい
次のコードは、`score = 50` のとき何を表示しますか。

```python
score = 50

if score >= 60:
    print("pass")
else:
    print("fail")
```
