### 演習 7

1. `for` の中で、その数字が偶数かどうかを確認します。
2. 偶数なら `print()` で表示します。

たとえば次のように書けます。

```python
numbers = [1, 2, 3, 4, 5]

for number in numbers:
    if number % 2 == 0:
        print(number)
```
