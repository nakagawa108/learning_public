# Python 中級 概要

## 1. この教材で学ぶこと
この教材では、Python の基本文法を学んだ次の段階として、少し実践的な書き方を学びます。

主に扱うのは次の 4 つです。

- イテレータとジェネレータ
- デコレータ
- コンテキストマネージャ
- 標準ライブラリの活用

どれも「難しい仕組み」ではありますが、実際には日常的なコードを読みやすく、安全に、短く書くための道具です。

### 要点
- 中級では「新しい文法を増やす」だけでなく、「よりよい書き方」を学びます。
- まずは道具の役割をつかむことが大切です。

## 2. 中級で最初につまずきやすい点
Python の入門では、変数、条件分岐、ループ、関数、クラスなどを学びます。

その次の段階では、次のような疑問が出やすくなります。

- `yield` は `return` と何が違うのか
- `@something` は何をしているのか
- `with open(...)` の `with` はなぜ必要なのか
- 標準ライブラリはどれを使えばよいのか

この教材では、そうした「見たことはあるが説明しにくいもの」を整理していきます。

### 要点
- 中級で大切なのは、見慣れない書き方を怖がらず意味で読めるようになることです。

## 3. 学習の流れ
この教材は次の順で進みます。

1. イテレータとジェネレータで、順番に値を取り出す考え方を学ぶ
2. デコレータで、関数に機能を追加する考え方を学ぶ
3. コンテキストマネージャで、開始と後片付けを安全にまとめる方法を学ぶ
4. 標準ライブラリで、実務でも使いやすい道具を知る

どの章でも、まず短い例を見てから説明し、最後に要点を確認します。

### 要点
- 学習順は「読み方を覚える」から「使いどころを知る」へ進みます。

## 4. 学ぶ前の前提
次の内容は知っている前提で進みます。

- `if`、`for`、関数定義、クラス定義の基本
- リスト、辞書、タプルの基本
- `open()`、例外処理、モジュール import を見たことがあること

もしこのあたりがまだ不安なら、無理に全部理解しきろうとせず、コード例の読み方に集中して進めて大丈夫です。

### 要点
- 完璧な前提知識は不要です。
- 「読める」「少し書ける」状態から中級へ進めます。

## 5. この教材のゴール
学習後には、次のようなコードを見たときに「何をしているか」を言葉で説明できることを目指します。

- `yield` を使った関数
- `@log_call` のようなデコレータ
- `with open(...)` や `with lock:` のような処理
- `pathlib.Path`、`Counter`、`itertools` を使ったコード

また、自分のコードでも「少し冗長な書き方」を、より Python らしい形へ書き換えられるようになることを目指します。

### 要点
- 目標は暗記ではなく、読めることと使いどころがわかることです。

## まとめ
- Python 中級では、実用的な書き方の道具を学びます。
- イテレータ、ジェネレータ、デコレータ、コンテキストマネージャ、標準ライブラリが中心です。
- 難しく見える記法も、役割ごとに分ければ理解しやすくなります。
