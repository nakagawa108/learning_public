# Python 中級

このテーマでは、Python の基本を終えた人向けに、一歩先の書き方を学びます。特にイテレータ、ジェネレータ、デコレータ、コンテキストマネージャ、標準ライブラリ活用を通して、読みやすく実用的な Python コードを書く感覚を身につけます。

## 対象者
- Python 入門はひと通り終えた人
- 実務や個人開発に向けて、もう少し Python らしい書き方を学びたい人

## 含まれる教材一覧
- `outputs/01_overview.md`: 学習全体の見取り図
- `outputs/02_lesson.md`: 重要概念とコード例つきの本編
- `outputs/03_exercises.md`: 易しい問題から始める演習
- `outputs/04_quiz.md`: 理解確認クイズ
- `outputs/05_answers.md`: 演習とクイズの解答

## 学習の進め方
1. まず overview を読んで、中級で何を学ぶのか全体像をつかみます。
2. 次に lesson を読み、短いコード例から少し実践的な例へ進みます。
3. exercises で手を動かし、quiz で理解を確認します。
4. 最後に answers を見て、自分の考え方と比較しながら弱点を補います。
