# Python 中級 教材設計

## テーマ名
Python 中級

## 想定読者
- Python の基本文法、関数、リスト、辞書、クラスの基本を学んだ人
- 入門は終えたが、少し実践的な書き方や標準ライブラリの使いどころを知りたい人

## 学習ゴール
- イテレータとジェネレータの役割を説明し、基本的なコードを書ける
- デコレータの基本形を読み書きできる
- コンテキストマネージャの意味を理解し、`with` を安全に使える
- `pathlib`、`collections`、`itertools` など中級者がよく使う標準ライブラリを活用できる
- 「少し Python らしい書き方」へ書き換える視点を持てる

## 教材出力物
- `README.md`
- `SKILL.md`
- `outputs/01_overview.md`
- `outputs/02_lesson.md`
- `outputs/03_exercises.md`

- `outputs/05_answers.md`

## 文体
- 基本文法を知っている学習者向けに、やさしく段階的に説明する
- 専門用語は最初に短く定義する
- 例を先に示し、そのあとに意味と使いどころを説明する
- 1 セクションで 1 つの概念に絞る

## 参照優先順位
1. `SKILL.md`
2. `README.md`
3. 既存の `outputs/`
4. 一般的な Python の基本知識

## 禁止事項
- 非同期処理、メタクラス、型システムの高度な話へ大きく広げすぎない
- 実装詳細や CPython の内部仕様を中心にしない
- 抽象的な説明だけで終わらせない
- 中級の入口として不要に難しい最適化話へ寄り道しない
