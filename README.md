# 学習教材集

各テーマごとに、概要、レッスン、演習、クイズ、解答をひととおりそろえた学習教材です。

## 含まれるテーマ

### [github-usage](./github-usage)
GitHub の使い方を初学者向けに学ぶ教材です。Git と GitHub の違い、リポジトリ、`clone`、`commit`、`push`、Pull Request の基本を順番に学べます。

主なファイル:
- `github-usage/README.md`
- `github-usage/outputs/01_overview.md`
- `github-usage/outputs/02_lesson.md`
- `github-usage/outputs/03_exercises.md`
- `github-usage/outputs/04_quiz.md`
- `github-usage/outputs/05_answers.md`

### [microsoft-agent-framework](./microsoft-agent-framework)
Microsoft Agent Framework を使った AI エージェント開発の基本を学ぶ教材です。Agent、Tools、Session、Memory、Workflow を段階的に整理しています。

主なファイル:
- `microsoft-agent-framework/README.md`
- `microsoft-agent-framework/outputs/01_overview.md`
- `microsoft-agent-framework/outputs/02_lesson.md`
- `microsoft-agent-framework/outputs/03_exercises.md`
- `microsoft-agent-framework/outputs/04_quiz.md`
- `microsoft-agent-framework/outputs/05_answers.md`
- `microsoft-agent-framework/source/notes.md`
- `microsoft-agent-framework/source/azure_openai_chat_client_sample.py`

## 学習の進め方
1. まず各テーマの `README.md` を読んで、何を学ぶ教材かを確認します。
2. 次に `outputs/01_overview.md` で全体像をつかみます。
3. `outputs/02_lesson.md` を読み、必要に応じてサンプルコードも確認します。
4. `outputs/03_exercises.md` と `outputs/04_quiz.md` で理解を試します。
5. 最後に `outputs/05_answers.md` を見て復習します。

## 想定している読者
- 新しい技術テーマを初学者向けに順番に学びたい人
- 説明だけでなく、例やサンプルコードも見ながら理解したい人
- 演習とクイズで理解を確認しながら進めたい人
