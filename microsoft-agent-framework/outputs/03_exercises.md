## 演習 7. Python 実装 1
### 問題
Python で最小のエージェントを作る練習です。次の条件を満たすサンプルを作ってください。

条件:
- エージェント名を `study_helper` にする
- instructions に「Python 学習の質問へ短く答える」ことを書く
- ユーザー入力として「Python のリストとは？」を渡す
- 1 回だけ実行して応答を表示する

### ヒント
- まずモデル用クライアントを用意する
- そのクライアントから agent を作る
- `run(...)` で入力を渡す

### ねらい
最小構成の流れを Python コードに落とす。

## 演習 8. Python 実装 2
### 問題
Python で簡単な Tool を使う練習です。次のような関数をツールとしてエージェントに渡す想定で、コードの骨組みを書いてください。

関数の仕様:
- 関数名は `get_weather`
- 引数は `city: str`
- 戻り値は文字列
- 例として `"Tokyo is sunny"` を返してよい

そのうえで、ユーザーが「東京の天気を教えて」と聞いたときに、このツールを使える形にしてください。

### ヒント
- まず Python の関数を書く
- その関数をエージェント側へ渡す
- 返答文そのものより、ツールを登録する流れを意識する

### ねらい
Tool を Python の関数として結びつける感覚をつかむ。
