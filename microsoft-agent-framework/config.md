# Microsoft Agent Framework 教材設計

## テーマ名
Microsoft Agent Framework

## 想定読者
- AI エージェント開発に初めて触れる初学者
- Python または C# の基本文法は少し読めるが、エージェント設計は未経験の人

## 学習ゴール
- Microsoft Agent Framework が何者かを説明できる
- Agent と Workflow の役割の違いを区別できる
- 最小のエージェント作成手順を理解できる
- Tools、Session、Memory、Workflow の使いどころを説明できる
- 公式ドキュメントを読みながら次の実装学習へ進める
